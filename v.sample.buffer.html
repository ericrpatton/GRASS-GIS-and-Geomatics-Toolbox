NAME

v.sample.buffer - Samples specified raster layers within a user specified distance from objects in a specified vector layer.

SYNOPSIS

v.sample.buffer
v.sample.buffer help
v.sample.buffer regionname=string vectfile=string buffersize=float rasters=string[,string,...] stats=string[,string,...]

Parameters:

regionname=string
Named region for geographic extent of region to survey
vectfile=string
Vector file to be used to survey region
buffersize=float
Size of buffer in map units around each object to survey
Default: 50.0
rasters=string
Rasters to be surveyed
stats=string
Statistics to be calculated from buffer area values
Options: mean, minimum, maximum, range, variance, standard_deviation, diversity, mode, mode_percent

DESCRIPTION

v.sample.buffer samples a GRASS raster or rasters in buffered areas surrounding objects in a vector layer. A series of summary statistics can be calculated for each location and these are added to the attribute table of the vector layer.

NOTES

This program was designed and tested for sampling areas around survey points. There is no reason however that it should not function for sampling and summarizing buffer zones around linear features or areas.
